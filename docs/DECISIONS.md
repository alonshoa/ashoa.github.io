# Platform and Localization Decisions

- **Hosting**: Stay on GitHub Pages with a Jekyll build pipeline.
- **Theme Migration**: Replace the existing Hacker theme with the Minimal Mistakes remote theme to unlock richer layouts, collections, and component overrides.
- **Localization**: Support both Hebrew (`he`) and English (`en`) content across core pages with mirrored URLs and a visible language toggle.
- **Content Pillars**: Organize the site around Projects, Teaching, Research, Writing, Demos, About, and CV sections as top-level navigation items.
- **Incremental Rollout**: Ship the migration in staged pull requests so content remains available while new layouts and collections come online.

> Tracking issue: create a repository issue referencing this document and keep it open until the bilingual Minimal Mistakes implementation is live.
