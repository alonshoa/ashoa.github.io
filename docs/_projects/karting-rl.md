Using Unity with ML-Agents, I trained a karting agent capable of navigating several race tracks. The project explores reward shaping, behavior cloning, and curriculum learning to help students reason about reinforcement learning in practice.

- **What I did:** Built training environment, tuned policies, documented the learning loop.
- **Key challenges:** Keeping training stable across tracks, exposing telemetry for classroom discussion.
- **Results:** Produced a demo showcased at GALA and reused in teaching materials.
